# Firebase Studio Desktop

A powerful Electron-based desktop application converted from a Firebase web app. 

## Features
- Seamless Firebase integration
- Built with Next.js and Tailwind CSS
- Desktop-native experience using Electron
- Contains AI flows with Genkit integration

## Prerequisites
- Node.js (v18 or higher recommended)
- Firebase project setup

## Installation

1. Clone this repository:
   ```bash
   git clone <your-repository-url>
   cd <repository-name>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Running the Application

### Development Mode

Run the Next.js frontend in development mode:
```bash
npm run dev
```

Run the Electron wrapper (in a separate terminal):
```bash
npm run electron
```

### Build for Production
To build the Next.js application:
```bash
npm run build
```
